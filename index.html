The study problem

AI can explain ochem. It just cannot draw it well.

ChatGPT, Claude, and other LLMs are patient and useful for walking through mechanisms, resonance, acid/base logic, and stereochemistry. The problem is that their chemistry representations often turn into ambiguous text, broken diagrams, or explanations that are hard to review later. ChemRender fixes the visual layer.

Tutorials

How to use ChemRender.

These are text walkthroughs for now. Illustrated tutorials with screenshots are coming later.

1. Set up and prompt your LLM

  1. Install ChemRender from the download section.
  2. Press the Copy Prompt button on this page to copy the ChemRender prompt.
  3. Paste that prompt into your preferred LLM (ChatGPT, Claude, etc.), then ask your organic chemistry question as usual.
  4. The LLM will answer with one or more molecule cards in the ChemRender format (blocks that start with ::mol).

2. Import cards from the clipboard

  1. Copy the LLM's reply (the text containing the ::mol blocks).
  2. In the ChemRender desktop app, press i to import the cards straight from your clipboard — no paste dialog.
  3. Each ::mol block renders as its own molecule card with lone pairs, charges, arrows, and captions.

The keyboard clipboard shortcuts work in the installed desktop app. In a browser, clipboard access is permission-gated, so the desktop app is recommended.

3. Replace a single card

  1. Select the card you want to update.
  2. Copy one valid molecule-card block to your clipboard.
  3. Press p to replace just the selected card with the clipboard content.

4. Understand the card format

Each card is a small text block. Atom references use element order across the card — O1 is the first oxygen, C2 the second carbon, and so on. 

::mol
title: Starting structure
smiles: O=C1[C-]CCCC1
lonepairs:
  O1: 2
  C2: 1
charges:
  C2: -
caption: One sentence about what to notice.
  • title — the card heading.
  • smiles — the molecule (standard SMILES).
  • lonepairs / charges — manual annotations by atom reference.
  • caption — a short, student-facing explanation.

5. Add curved arrows

Add an arrows section to show electron movement. Endpoints reference atoms, lone pairs, or bonds.

arrows:
  O1.lp1 -> N1-H1 curve: left
  N1-H1 -> N1 curve: right
  • O1.lp1 — the first lone pair drawn on O1.
  • O1 — the atom O1.
  • C1-O1 — the bond between C1 and O1.
  • curve: left / curve: right — optional bend direction.

6. Export a card as an image

  1. Open Profile and enable Show SVG/PNG export buttons.
  2. Use the per-card SVG button for a clean vector download.
  3. Use the per-card PNG button for a high-resolution image.

Exports include the molecule drawing and its annotations on an off-white background. They do not include the app UI, title, or caption.
