ChemRender changelog

A running history of notable ChemRender updates, generated from the app changelog.

Notable changes to Text to Chem, reconstructed from git history. This repository does not currently use release tags, so entries are grouped by commit date.

0.1.2 - 2026-06-13

Added

  • Added a centralized settings system grouped into Rendering, Cards, Import & Packages, Appearance, Keyboard, Help & Docs, and Advanced, with a settings search.
  • Added global and per-card zoom, plus a three-dots card menu for copy, export, zoom, replace, and delete.
  • Added a configurable header (keep open or auto-hide), card density, caption visibility, and reduced-motion options.
  • Added a Copy dropdown that builds the AI prompt from parts (base format, arrows, stereochemistry, radicals).
  • Added signed and notarized macOS release tooling, including a local signing kit and a desktop:sign script.

Changed

  • Simplified molecule cards into a passive reader: removed card numbers and the selected-card model, and made j/k scroll the page vertically.
  • Reworked the top bar to two core actions, Import and Copy (the AI prompt), alongside a settings gear.
  • Moved per-card actions behind a hover/focus Details button and a three-dots menu instead of an always-visible action row.
  • Made the Import button run the configured workflow (clipboard or paste box, replace or add) in one click.
  • Made package tabs optional and hidden by default.
  • Kept the settings dialog at a fixed height so it no longer resizes when switching sections.

Removed

  • Removed settings that did not yet affect rendering (formal-charge, atom-label, and functional-group toggles, molecule size, key hints, and navigation feedback) so the settings stay honest.
  • Removed the separate prompt-guide buttons from the top bar in favor of the Copy dropdown.

Fixed

  • Fixed per-card zoom so the molecule drawing actually scales.

0.1.1 - 2026-06-12

Added

  • Added stereochemistry descriptor support, including stereo descriptor parsing/rendering and focused tests.
  • Added prompt, settings-store, SVG export, toast, and app-template modules as part of the main application split.
  • Added tests for stereo parsing, stereo descriptors, and expanded parser behavior.

Changed

  • Modularized the large main.js application entry point into smaller source modules.
  • Updated renderer, annotations, parser, keybinding, and export behavior to support the new stereo workflow.
  • Updated Tauri and package metadata for the current desktop app build.

2026-06-03

Changed

  • Changed tutorial links to open on the public website instead of inside an in-app tutorial modal.
  • Set the browser tab title to ChemRender.

2026-06-02

Added

  • Added a Windows installer build script.
  • Added .env.local to ignored local environment files.

Changed

  • Rebranded desktop packaging from Text to Chem toward Chem Render naming.
  • Updated macOS release script and Tauri configuration for the rebrand.

2026-06-01

Added

  • Added the Tauri desktop app release path with native clipboard support, macOS release workflow, release script, Tauri configuration, and release documentation.
  • Added clipboard import/replace handling for desktop clipboard workflows.
  • Added keybinding settings and tests.
  • Added functional group detection patterns and highlighting support.

Changed

  • Collapsed and refined the top toolbar/chrome behavior, including auto-hide fixes.
  • Removed the earlier GitHub Pages deployment path from the app repository.
  • Updated release and macOS distribution documentation.

Fixed

  • Fixed duplicate functional group detection.

2026-05-30

Added

  • Added vim-style keyboard navigation and richer keyboard-driven card workflows.
  • Added card navigation feedback options, including off, left, middle, and right positions.

Changed

  • Improved the main UI, renderer integration, annotations, color theme, and card interaction styling.
  • Made card navigation feedback configurable and off by default.

2026-05-29

Added

  • Added privacy-preserving GoatCounter usage analytics behind configuration.
  • Added GitHub Pages release scaffolding, service worker, web app manifest, and public icons.
  • Added SVG/PNG export controls for molecule cards.
  • Added project license file.

Changed

  • Restored the package import workflow.
  • Polished the local renderer workflow, demo content, export controls, top bar, logo, favicon, and README/deployment notes.
  • Updated GitHub Pages automation to use Node 24.

2026-05-28

Added

  • Created the initial Text to Chem local renderer for structured molecule-card notes.
  • Added parsing for ::mol blocks, SMILES rendering, lone pairs, formal charges, captions, atom references, and curved-arrow annotations.
  • Added the initial Vite app structure, renderer, annotation, arrow, parser, theme, and stylesheet modules.
  • Added parser and arrow workflow tests.

Changed

  • Stabilized parser and arrow workflows after the initial implementation.
  • Polished the local renderer workflow.
